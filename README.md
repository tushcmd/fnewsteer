# FNEWSTEER - Fundamental Analysis(News) Steer

## Description

Simple tool for price action traders to avoid steering blindly in the markets.
Avoid unexpected movements in price caused by news events despite having done good technical analysis.

## Who this is for

For you, fundamental/news analysis is not significant as a profit-generating model, but it is 100% significant as a loss-prevention model. You don't need to predict the news; you need to predict how the candle looks 30 minutes after the news.

Without it:
"You might be a great surfer, but you paddle out into a hurricane unaware". - Unknown.
